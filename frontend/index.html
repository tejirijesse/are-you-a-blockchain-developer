Prove the ALU logo is authentic — on-chain.

This decentralized app registers the official ALU logo on a blockchain using its SHA-256 content hash, lets anyone verify a logo file against the on-chain record, and tracks ownership shares through an ERC-20 token.

Part B

Register Logo

Upload the logo, hash it in your browser, and mint an on-chain asset record with its SHA-256 fingerprint.
Part C · No wallet needed

Verify Logo

Check any logo file (or a pasted hash) against the on-chain record to confirm it has not been modified.
Part D

Token Dashboard

View total supply, your ownership percentage, and — if you are the owner — distribute ALUT ownership shares.

How it works

Content hashing

The logo file is read and hashed with SHA-256 directly in your browser using the Web Crypto API. The 32-byte hash is stored on-chain as a bytes32 fingerprint — the file itself never leaves your device.

Ownership shares

1,000,000 ALUT tokens represent ownership of the registered brand asset. The contract owner can distribute shares, and anyone can read their ownership percentage from the ERC-20 contract.
